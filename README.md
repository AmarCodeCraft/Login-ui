# React Login Form

This is a simple **Login Form project built with React and Tailwind CSS**.
It demonstrates basic React concepts like **useState**, form handling, and simple UI styling.

---

## 🚀 Features

* Simple Login UI
* React useState Hook
* Tailwind CSS styling
* Centered login card
* Gradient background

---

## 🛠️ Technologies Used

* React
* Tailwind CSS
* JavaScript
* HTML

---

## 📂 Project Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/AmarCodeCraft/Login-ui.git
```

### 2️⃣ Go to project folder

```
cd your-repository-name
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the project

```
npm start
```
